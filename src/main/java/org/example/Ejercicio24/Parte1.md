**Punto 1: Unión e intersección**

**Union:** La unión de dos o más conjuntos da como resultado un conjunto completamente nuevo que contiene una combinación de elementos presentes en ambos conjuntos.

**Intersección:** El conjunto de todos los elementos del conjunto A que también están presentes en el conjunto B. 

Fuente: https://www.cuemath.com/algebra/

**Punto 2: Pre y post condiciones**

_Precondición:_
* Conjuntos ordenados
* No hay elementos repetidos

_Postcondición:_
* El conjunto resultado contiene todos los elementos de ambos conjuntos
* No hay elementos repetidos
* Ordenado
